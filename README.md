# iMessage-Sync-Shortcut

A utility designed to instantly trigger an iCloud sync for Apple Messages on iOS and macOS. This tool was created to solve sync delays caused by high message volumes, allowing to force a refresh without navigating through multiple layers of system menus.

---

| iOS | MacOS |
| ----------- | ----------- |
| Deep-links (usually obfuscated and were a pain to find) directly to iCloud Settings, saving multiple taps. | Uses AppleScript (GUI Scripting) to automatically click through the Messages app preferences to initiate a sync. (Requires macOS Monterey+)  |


[iCloud Link](https://www.icloud.com/shortcuts/d5f5ee64e0b64e559d50b914dff80174) to Shortcut.
