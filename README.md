# iMessage-Sync-Shortcut
---

| iOS | MacOS |
| ----------- | ----------- |
| A quick link to *iMessage Sync* found under Settings > iCloud Settings... (Does not select GUI elements) | Runs Applescript to click through GUI elements to intialize sync in *Message* app. (Monterey or newer)  |
